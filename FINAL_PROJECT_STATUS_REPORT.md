## 🚀 使用指南

### 快速开始

```bash
# 1. 克隆项目
git clone https://github.com/your-org/webassembly-rust.git
cd webassembly-rust

# 2. 设置开发环境
.\scripts\setup-dev.ps1 -Full

# 3. 构建项目
.\scripts\build.ps1

# 4. 运行测试
.\scripts\build.ps1 -Test

# 5. 启动前端演示
cd frontend
npm install
npm start
```
